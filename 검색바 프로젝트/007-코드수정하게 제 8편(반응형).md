# 반응형 적용 내용

## 반응형 적용 코드

데스크탑 기준으로 media를 적용하였으며 `@mixin`을 이용하여 다른 `scss`에서도 사용가능하도록 수정하였습니다.

```scss
@mixin mobile {
  @media screen and (max-width: 767px) {
    @content;
  }
}

@mixin tablet {
  @media screen and (max-width: 1023px) {
    @content;
  }
}

@mixin desktop {
  @media screen and (min-width: 1024px) {
    @content;
  }
}
```
