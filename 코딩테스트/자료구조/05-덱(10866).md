# 덱 (10866)
**덱: 양쪽 끝에서 삽입과 삭제가 모두 가능한 자료 구조의 한 형태이다.   
두 개의 포인터를 사용하여, 양쪽에서 삭제와 삽입을 발생시킬 수 있다. 큐와 스택을 합친 형태로 생각할 수 있다.**

|:---:|:---:|:---:|:---:|
|덱|10866|nodejs|[바로가기](https://www.acmicpc.net/problem/10866)|

## 문제 풀이
1. 덱 규현
2. 명령어
  - `push_front`: 앞에 저장
  - `push_back`:  뒤에 저장
  - `pop_front`:  앞에 삭제
  - `pop_back`:  뒤에 삭제
  - `size`:  크기
  - `empty`:  비어있으면 1 아니면 0
  - `front`: 앞에 출력 ?? -1
  - `back`:  뒤에 출력 ?? -1
3. N 범위 1 ≤ N ≤ 10,000
